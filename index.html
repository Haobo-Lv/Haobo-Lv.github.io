Large-scale image search and classification

Learning Binary Codes for Fast Similarity Search

This project investigates novel learning methods to learn binary codes for high dimensional data. We have developed efficient optimization methods for generating binary codes, as well as learning methods for learning data-dependent binary codes. It was applied to large-scale image retrieval applications with tens of millions of data points.

Large-scale ranking (ongoing)

Developing large-scale ranking methods for modeling Internet images and sentences. Use approximated ranking methods to learn the relationship between large amount of Internet images and the meaning of their corresponding sentences/tags.
